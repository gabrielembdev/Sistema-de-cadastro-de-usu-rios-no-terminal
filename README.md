# Sistema-de-cadastro-de-usu-rios-no-terminal
Projeto simples desenvolvido em C# com .NET para praticar lógica de programação e conceitos básicos da linguagem.

 Sobre o projeto

O sistema funciona no console e permite gerenciar usuários informando nome e idade. Os dados são armazenados em uma lista durante a execução do programa.

 Funcionalidades

Cadastrar usuário

Listar usuários cadastrados

Atualizar dados de um usuário

Remover usuário

Menu interativo no console

 Conceitos praticados

Classes e objetos

Listas (List<T>)

Métodos

Estruturas de repetição

Controle de fluxo (switch)

Entrada de dados com Console.ReadLine

🛠 Tecnologias

C#

.NET

Aplicação Console

🎯 Objetivo

Praticar os fundamentos de programação e compreender a estrutura básica de um CRUD (Create, Read, Update, Delete).
